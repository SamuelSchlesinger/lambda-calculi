# Church 1932, 1933 -- "A Set of Postulates for the Foundation of Logic"

## Citation

- Church, A. (1932). "A Set of Postulates for the Foundation of Logic." *Annals of Mathematics*, 33, 346--366.
- Church, A. (1933). "A Set of Postulates for the Foundation of Logic (Second Paper)." *Annals of Mathematics*, 34, 839--864.

## Summary

These two papers introduced the lambda calculus as part of a broader formal system
intended to serve as a foundation for all of mathematics. Church began developing
the system in 1928--29, hoping to build a type-free system in which all variables
occurred only bound, avoiding contradiction by restricting the law of excluded middle.

The 1932 paper introduced lambda notation for function abstraction and application.
Shortly after publication, Church discovered the axiom set was contradictory.
The 1933 paper modified the axioms to attempt to avoid the contradiction.

Both systems were subsequently shown to be inconsistent by Kleene and Rosser in 1935,
who demonstrated that Richard's paradox could be derived within them. However, Church
isolated the purely functional fragment -- the pure lambda calculus -- and proved
its consistency in 1935.

## Key Contribution

First introduction of lambda-abstraction notation and the fundamental concepts of
function abstraction and application that constitute the lambda calculus.
