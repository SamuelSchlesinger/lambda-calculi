# Pfenning and Paulin-Mohring: Inductively Defined Types in the Calculus of Constructions (1989/1990)

## Citation
Pfenning, F. and Paulin-Mohring, C. "Inductively defined types in the Calculus
of Constructions." In Mathematical Foundations of Programming Semantics (MFPS 1989).
LNCS 442. Springer, New York, 1990.

## Key Contributions
- Defined the notion of an inductively defined type in the Calculus of Constructions
- Showed how inductively defined types can be represented by closed types
- Demonstrated that all primitive recursive functionals over these inductively defined
  types are also representable
- Systematic derivation of elimination principles from constructor specifications

## Technical Approach
- Inductive types characterized by constructor specifications
- Elimination rules (recursors) systematically derived
- Universe hierarchy respected, preserving predicativity constraints
- Foundation for the implementation in Coq
