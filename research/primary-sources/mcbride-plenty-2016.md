# McBride, C. (2016). I Got Plenty o' Nuttin'.

## Publication
In: A List of Successes That Can Change the World -- Essays Dedicated to
Philip Wadler on the Occasion of His 60th Birthday. Springer LNCS.

## Summary

Introduces usage annotations in dependent type theory that form the basis for
Quantitative Type Theory. Proposes tracking how many times each variable is used
via semiring annotations.

## Key Ideas

- Variables annotated with usage information from a semiring
- 0 usage = erased at runtime (irrelevance)
- 1 usage = used exactly once (linearity)
- omega usage = unrestricted use
- Dependent types and linear types unified in a single framework
- Foundation for Atkey's later formal development of QTT
- Foundation for Idris 2's type system
